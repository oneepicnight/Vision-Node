# Vision Node v1.0 - Windows Mainnet Release

## Release Date
January 6, 2026 (Launch Day)

## Package Contents
- vision-node.exe - Optimized mainnet binary (~34 MB)
- start.bat - Quick launcher script
- config/external_rpc.json - GetBlock.io BTC/BCH deposit integration
- public/ - Web wallet and dashboard UI
- keys.json.example - Node identity configuration template
- p2p.json - P2P network configuration
- miner.json - Mining configuration
- README.md - Project documentation
- NOTICE - Legal notices

## Quick Start

### First Time Setup
1. Copy keys.json.example to keys.json
2. Edit keys.json with your node configuration
3. Double-click start.bat

### Accessing the Interface
- Web Wallet: http://localhost:7070/app
- Dashboard: http://localhost:7070/panel.html
- Mining Panel: http://localhost:7070/panel.html
- API: http://localhost:7070/api/

## System Requirements
- Windows 10/11 or Windows Server 2019+
- 8GB RAM minimum (16GB recommended)
- 50GB free disk space for blockchain data
- Stable internet connection for P2P sync

## Security Notes
 This is production mainnet software
- Keep your keys.json file secure and backed up
- Configure firewall to allow P2P port (default 7072)
- Do not share your private keys
- Regularly backup your blockchain data

## Data Directory
The node stores blockchain data in:
- Windows: .\vision_data_7070\

## Firewall Configuration
If running a public node, allow inbound connections on:
- Port 7070 (HTTP API/Web UI)
- Port 7072 (P2P network)
