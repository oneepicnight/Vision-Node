# Vision Node v1.0.3 - Windows Mainnet Release

## Release Date
January 11, 2026

## What's New in v1.0.3
🔧 **Critical Fixes:**
- **Semantic Version Comparison**: Fixed peer compatibility checks using proper semantic versioning instead of string comparison
  - Now correctly handles version formats: v1.0.3, 1.0.3, 1.0, etc.
  - Fixes issue where "1.0.0" < "v1.0.0" incorrectly rejected compatible peers
- **P2P Port Fix**: Outbound peer connections now always use port 7072 (P2P) instead of incorrectly defaulting to 7070 (HTTP)
- **Enhanced Compatibility Logging**: Added detailed diagnostics showing why peers are marked incompatible

## Previous Fixes (v1.0.1)
🔧 **Critical Fixes:**
- Fixed miner rewards not being distributed for peer-received blocks
- Fixed tokenomics application in block acceptance and reorg paths
- All blocks now properly credit emission, tithe, and fee distributions

✨ **New Features:**
- Command Center dashboard for real-time node monitoring
- Mini Command Center widget on all pages
- Enhanced mining status indicators
- Guardian status tracking and visualization
- System mood/health monitoring
- Real-time event logging

## Quick Start

### First Time Setup
1. Copy keys.json.example to keys.json
2. Edit keys.json with your node configuration
3. Double-click start.bat

### Accessing the Interface
- **Command Center**: http://localhost:7070/app (login, then click Command Center)
- Web Wallet: http://localhost:7070/app
- Dashboard: http://localhost:7070/panel.html
- Mining Panel: http://localhost:7070/panel.html
- API: http://localhost:7070/api/

## Command Center
The new Command Center provides a comprehensive real-time overview:
- **Node Status**: Connection state, block height, sync progress
- **Mining Status**: Active/inactive, hashrate, blocks mined
- **Wallet Balances**: LAND, CASH, and GAME token balances
- **Guardian Status**: Connection to beacon, role information
- **System Mood**: Network health and node performance metrics
- **Event Log**: Real-time system events and notifications

Access via the web wallet at http://localhost:7070/app → Command Center

The Mini Command Center appears at the top of every page showing:
- Node online/offline status with block height
- Mining status and recent blocks found
- Current wallet balances
- Guardian connection status

Click the Mini Command Center to open the full dashboard.

## Security Notes
 This is production mainnet software
- Keep your keys.json file secure and backed up
- Configure firewall to allow P2P port (default 7072)
- Do not share your private keys
- Regularly backup your blockchain data

## Data Directory
The node stores blockchain data in:
- Windows: .\vision_data_7070\

## Firewall Configuration
If running a public node, allow inbound connections on:
- Port 7070 (HTTP API/Web UI)
- Port 7072 (P2P network)
