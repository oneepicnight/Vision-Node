========================================
Vision Node - Integrated Release
========================================

This package includes:
- Vision Node executable
- Integrated Wallet (Web UI)
- Miner Panel (Web UI)
- Dashboard (Web UI)

QUICK START:
------------
1. Double-click "START-VISION-NODE.bat" to start the node
   OR
   Run: vision-node.exe

2. Access the interfaces:
   - Wallet:       http://127.0.0.1:7070/app
   - Miner Panel:  http://127.0.0.1:7070/panel.html
   - Dashboard:    http://127.0.0.1:7070/dashboard.html

SYSTEM REQUIREMENTS:
-------------------
- Windows 10/11
- 4GB RAM minimum
- 1GB free disk space

FEATURES:
---------
✓ Integrated wallet - no separate installation needed
✓ Miner panel for mining control
✓ Real-time blockchain dashboard
✓ P2P networking
✓ Full node capabilities

DATA STORAGE:
-------------
Blockchain data is stored in: ./vision_data_7070/
To reset the blockchain: Delete this folder and restart

CONFIGURATION:
--------------
Default port: 7070
To change port: Set environment variable VISION_PORT=<port>

SUPPORT:
--------
Issues? Check the logs in the console window where vision-node.exe is running.
