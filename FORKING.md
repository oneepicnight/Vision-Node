# Forking & Trademark Policy

- You may fork this repository under the Apache 2.0 license (see [LICENSE](LICENSE)).
- Do **not** use the "Vision" name, logo, or associated marks for derivative networks or products without written permission.
- If you run a public fork, clearly differentiate network IDs, bootstrap checkpoints, and branding to avoid user confusion.
- Remove or override default telemetry endpoints and admin branding when creating a fork.
- Keep security disclosures for this upstream project private; forks are responsible for their own security process.
